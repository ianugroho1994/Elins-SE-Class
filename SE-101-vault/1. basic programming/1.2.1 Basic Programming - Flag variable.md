# Konsep 3.5 - Flag
- in the nustshell : variable Yes / No atau True / False
- berfungsi untuk dipakai di decision sebagai penanda kapan satu if block di esekusi
- Flag variable sederhana
	- berbentuk boolean datatype
	- menggunakan penamaan is.....
- Flag dengan enum
	- berbentuk variable enum / enumarable / enumarated
	- bisa terdiri dari beberapa state, tidak hanya true false
- kegunaan
	- untuk bug fix
	- untuk feature flag
	- untuk state machine
## kesalahan umum
- menggunakan negative statement
- menggunakan flag selain boolean datatype
- tidak menginisialisasi nilai enum
- menambahkan enum baru dari tengah
